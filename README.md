Gym Member Exercise Analysis Dashboard

Project Overview

This Power BI project provides a data-driven exploration of gym member performance and health metrics. By analyzing a dataset of 1,000 members, the dashboard uncovers critical correlations between workout types, body composition (BMI/Fat %), and physiological responses such as heart rate and calorie expenditure.

Dashboard Preview

This single-page comprehensive dashboard integrates demographic insights with performance biometrics to provide a 360-degree view of member health.

Key Analytical Insights

Energy Expenditure: Identified Cardio as the most efficient workout type for high caloric burn.

Biometric Correlation: Visualized the relationship between Water Intake and Resting BPM, highlighting the importance of hydration in cardiovascular recovery.

Body Composition: Analyzed BMI distribution across different experience levels, showing how veteran members maintain optimized physical metrics.

Intensity Tracking: Real-time monitoring of Average vs. Max BPM across diverse activities like Yoga, HIIT, and Strength training.

Technical Implementation

Tool: Power BI Desktop

Data Modeling: Implemented a clean data schema for efficient querying of synthetic gym member records.

DAX Measures: Developed custom DAX expressions for calculated metrics, including average intensity scores and calorie-to-duration ratios.

Power Query: Performed extensive data cleaning, including type transformations and the creation of conditional BMI categories.

Dataset Attributes

The analysis is built upon the following key data points:

Demographics: Age, Gender.

Physicals: Weight (kg), Height (m), BMI, Fat Percentage.

Session Data: Workout Type, Duration, Frequency, Experience Level.

Health Biometrics: Avg/Max/Resting BPM, Calories Burned, Water Intake.
